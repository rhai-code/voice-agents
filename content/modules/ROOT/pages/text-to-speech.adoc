Deploy the Higgs-Audio Model using RHOAI and Deployment.

== Deployment

Deploy the Higgs-Audio model using the following command:

[source,bash,subs="attributes+"]
----
oc apply -f models/higgs-audio/higgs-audio-v2-deployment.yaml
----

== Testing

Test the model with some text:

[source,bash,subs="attributes+"]
----
curl -X POST $MODEL_URL/v1/audio/speech \
  -H "Content-Type: application/json" \
  -d '{
    "model": "higgs-audio-v2-generation-3B-base",
    "voice": "belinda",
    "input": "What would you like on your pizza?",
    "response_format": "pcm"
  }' \
  --output - | ffmpeg -f s16le -ar 24000 -ac 1 -i pipe:0 -f wav - | ffplay -nodisp -autoexit -
----

This command:

1. Sends a text prompt to the TTS model
2. Receives PCM audio data
3. Converts it to WAV format using ffmpeg
4. Plays it directly using ffplay

The Higgs-Audio model generates natural-sounding speech from text, completing the voice agent loop.
