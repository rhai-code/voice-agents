== Server Side

In your voice notebook.

Install requirements.

[source,bash]
----
uv pip install -r requirement.txt
----

Run web socket server in a notebook shell.

[source,bash]
----
python ws_server.py
----

image:server-side.png[Server Side]

== Client Side

[source,bash]
----
cd web
----

Install requirements.

[source,bash]
----
npm i
----

Build a production version.

[source,bash]
----
npm run build
----

Run the Web UI in a notebook shell.

[source,bash]
----
npx next dev -H 0.0.0.0 -p 3000
----

image:client-side.png[Client Side]

Apply client side routing.

[source,bash]
----
oc apply -f web-ui-routing.yaml
----

Browse to the Web UI Route.
