== Overview

The Pizza Shop application demonstrates a real-world voice agent that can:

* Take pizza orders through natural conversation
* Handle pizza type selection (vegetarian, meat lovers, etc.)
* Calculate order totals
* Arrange delivery details
* Manage conversation state and interruptions
