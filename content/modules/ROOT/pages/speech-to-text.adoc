= Speech to Text
:source-highlighter: rouge
:toc: macro
:toclevels: 2

Deploy the Whisper Model using RHOAI and LLM Inference.

== Deployment

Deploy the Whisper model using the following command:

[source,bash]
----
oc apply -f models/whisper/whisper-llmisvc.yaml
----

== Testing

Test the model with an audio file:

[source,bash]
----
curl -s -X POST ${MODEL_URL}/v1/audio/transcriptions \
  -H "Content-Type: multipart/form-data" \
  -H "Authorization: Bearer ${ACCESS_TOKEN}" \
  --form file=@/home/mike/Downloads/hello.wav \
  --form model=whisper | jq .
----

== Example Response

[source,json]
----
{
  "text": " Hello.",
  "usage": {
    "type": "duration",
    "seconds": 3
  }
}
----

The Whisper model converts speech audio into text, which is the first step in our voice agent pipeline.
